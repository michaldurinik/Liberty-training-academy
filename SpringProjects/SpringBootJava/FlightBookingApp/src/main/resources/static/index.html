The Flight Booking Application

This is a prototype application for booking flights, built using Spring MVC. The basic functionality is as follows:

  1. On the first page you enter where you want to fly from and to, with the associated dates
  2. You are then presented with all the matching flights, and choose which ones you want
  3. Finally you are presented with a confirmation or failure page.

The packages used in the code are presented below:

PackageDescription
demos.spring.boot.flights.controllers Contains controller classes that are associated with URL's and respond to GET or POST requests (or both)
demos.spring.boot.flights.domain and demos.spring.boot.flights.services Contain the core flight booking functionality, this is drawn from an exercise you did on the Spring course
demos.spring.boot.flights.beans Contains basic JavaBeans that are used to store information entered into HTML forms
demos.spring.boot.flights.domain.errors Contains application specific exception classes and a global exception handler
